# gif-streaming    
gif-streaming は、主にエッジコンピューティング環境においてUIフロントエンドにgif画像を送信するマイクロサービスです。      

## 動作環境  
gif-streaming は、Kubernetes および AION 上での動作を前提としています。 
以下の環境が必要となります。  
・OS: Linux OS  
・CPU: ARM/AMD/Intel  
・Kubernetes  
・AION  

## gif画像について  
本リポジトリにはgif画像は含まれておりません。   
gifsフォルダに任意のgif画像を入れてお使いください。
